# Changelog

## 1.0.0 - 2026-06-18

- Added a modern public home page, login page, dashboard, and report screens.
- Added README documentation with screenshots and local run instructions.
- Migrated application configuration to YAML with environment-variable based secrets.
- Added production profile settings for safer database validation, disabled SQL logging, and secure session cookies.
- Added admin bootstrap configuration for first-run setup.
- Added common exception handling and user-friendly error pages.
- Refactored controllers, services, DTO mappers, and security configuration for cleaner code.
- Added PDF reports, Excel imports, and PostgreSQL-backed persistence.
